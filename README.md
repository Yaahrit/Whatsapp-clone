# WhatsApp Web Clone (Modernized)

A premium, fully interactive WhatsApp Web clone built with pure HTML, CSS, and Vanilla JavaScript. This project transforms a static UI into a functional chat application with real-time-like behavior.

## ✨ Key Features

- **Interactive Chat System:** Open specific chats dynamically with updated headers and message histories.
- **Real-Time Messaging:** Send messages instantly with smooth slide-up animations and auto-scroll behavior.
- **Smart Auto-Reply:** Simulated "typing..." status followed by automated intelligent responses.
- **Dynamic Chat List:**
  - Automated reordering (active chats move to top).
  - Live message previews and timestamp updates.
  - Interactive unread badges with pulse animations.
- **Premium UI/UX:**
  - Official WhatsApp Web color palette and Segoe UI typography.
  - Smooth CSS transitions and micro-animations.
  - Responsive "app-like" experience within a fixed container.
- **Tab Navigation:** Seamless switching between Chats, Status, and Calls using pure CSS logic.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and dynamic DOM elements.
- **CSS3:** Custom variables, Flexbox/Grid, and advanced keyframe animations.
- **JavaScript (ES6+):** State management, real-time DOM manipulation, and simulation logic.
- **IonIcons:** Modern iconography for a clean, professional look.

## 📁 Project Structure

```text
whatsapp-clone/
├── css/
│   └── whatsapp.css   # Core styling and animations
├── js/
│   └── script.js     # Chat logic and interactions
├── images/           # User avatars and assets
├── index.html        # Main entry point
└── README.md         # Project documentation
```

## 🚀 How to Run

1. Clone the repository or download the source code.
2. Ensure you have the `images` folder with the necessary assets.
3. Open `index.html` in any modern web browser.
4. Click on any contact to start chatting!
